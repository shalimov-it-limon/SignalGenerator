SignalGenerator — приложение для генерации и анализа математических сигналов (синусоида и меандр).

Системные требования:
	 .NET 5.0;
	 Windows / Linux / macOS;
	 Visual Studio 2019

Пример работы (консоль):
Выберите тип сигнала: 1 - Синусоида, 2 - Меандр
1
Амплитуда: 5
Частота: 2
Количество точек: 1000

Max: 4.999
Min: -4.999
Avg: 0.001
Zero crossings: 4
Файл сохранен.

Формат файла:
Type: sine, Amplitude: 10, Frequency: 100
Time,Value
0,0
0,0033333333333333335,8,660254037844386
0,006666666666666667,-8,660254037844389
...

Тест-кейсы

TC-001
Название: Генерация синусоиды
Предусловия: Приложение запущено
Шаги:
1.	Выбрать тип 1
2.	Ввести амплитуду 5
3.	Частоту 2
4.	Точки 1000
Ожидаемый результат:
Отображаются корректные значения max/min/avg, файл создан.

TC-002
Название: Ошибка при неверном количестве точек
Шаги:
Ввести 50
Ожидаемый результат:
Сообщение: "Количество точек должно быть от 100 до 10000"


TC-003
Назвыание: Сохранение файла
Шаги: 
1. Создать точку сигнала
2. Сохранить результат в файл
Ожидаемый результат: 
Файл успешно создан.
